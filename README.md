We developed an end-to-end Plant Disease Detection System using Deep Learning to automatically identify 38 distinct plant disease classes from leaf images. Crop diseases result in nearly 20% to 40% loss in global yields annually. To counter this, we implemented state-of-the-art methodology based on a 2025 MDPI paper by Krishna et al. , scaling their methods onto the PlantVillage dataset consisting of over 54,000 images. We engineered a full production pipeline: from Exploratory Data Analysis (EDA) and data augmentation to an optimized EfficientNet-B3 model , and deployed it as a real-time Streamlit web application that offers both disease predictions and actionable treatment suggestions.
